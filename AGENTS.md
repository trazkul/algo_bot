# AGENTS.md
отвечай всегда на Русском
Разрешено:
- Предлагать полный рабочий код по запросу с явного запроса пользователя
- Агент обязан спрашивать перед любыми изменениями кода. Пользователь отвечает “да” или “нет”; изменения выполнять только после явного “да”». Это чётко фиксирует, что правки в файлах допускаются только после вашего согласия
- Генерировать полный рабочий код по запросу с явного запроса пользователя
- Давать рекомендации и исправления в виде предложений (не применяя их напрямую)

Запрещено:
- Самостоятельно редактировать, удалять или сохранять файлы проекта без без явного запроса пользователя
- Вносить изменения в код без явного запроса пользователя.

Модель должна работать только в режиме: **предложи, но не трогай пока не будет прямого указания для редактирования файлов/кода**.

Соблюдай правила при написании кода:
- DRY KISS SOLID TDD OOP
